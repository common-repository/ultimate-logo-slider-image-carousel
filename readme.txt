== Description ==
</h1>Wordpress' easiest Logo Slider & Carousel</h1>

A fantastic plugin and interface for displaying & managing multiple logo sliders and carousels

Ultimate Logo Slider & Image Carousel is an easy to use interface that helps you display multiple logos & images in a modern and sleek logo slider design on your website. 

Use the given shortcode to display the logo slider anywhere on your site.

<h3>Key features</h3>
<ul>
<li>Control multiple logo sliders with different shortcodes allowing you to styles to each one specifically to a page or post</li>
<li>Hundreds of customisability options</li>
<li>Unlimited shortcodes</li>
</ul>
